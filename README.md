# Как запустить?

## Чтобы запустить игру, запустите файл game

Склонируйте репозиторий;
Запустите игру (файл game.py запускает все);
Следуйте инструкциям в консоли;
Создайте аккаунт или войдите.
Выберите "new game" или "continue"
Исследуйте локации, сражайтесь, собирайте артефакты!

Все данные сохраняются автоматически после каждой локации.
Код написан с акцентом на модульность и расширяемость: легко добавить новые классы, локации, заклинания или врагов.
